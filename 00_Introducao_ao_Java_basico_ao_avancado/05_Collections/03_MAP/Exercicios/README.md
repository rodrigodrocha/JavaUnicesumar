# Exercício: Utilizando Map para Controle de Estoque
<img src="..\..\01_SET\exercicio\image.png"></img>
## Objetivo
O objetivo deste exercício é entender e aplicar conceitos de coleções em Java, especificamente o uso de ``Map`` para gerenciar um estoque de produtos.

## Instruções
1. Crie um programa que utilize um ``Map`` para representar o estoque de produtos.

2. Adicione produtos ao estoque com códigos e quantidades.

3. Implemente a verificação da quantidade disponível de um produto específico.

4. Implemente a remoção de uma quantidade específica de um produto do estoque.

## Passos para Fazer o Exercício
1. Importe as classes necessárias:

    * Utilize ``HashMap`` e ``Map`` da biblioteca ``java.util``.

2. Crie o método ``main``:

    * Inicialize um ``Map`` para representar o estoque de produtos.

    * Adicione produtos ao estoque com códigos e quantidades.

    * Verifique a quantidade disponível de um produto específico.

    * Remova uma quantidade específica de um produto do estoque e atualize o estoque.

3. Teste a funcionalidade:

    * Execute o programa e verifique se as operações de adição, verificação e remoção estão funcionando corretamente.

# Exercício: Utilizando List para Manipular Lista de Filmes

## Objetivo
O objetivo deste exercício é entender e aplicar conceitos de listas em Java, utilizando a classe ``ArrayList`` para gerenciar uma lista de filmes.

## Instruções
1. Crie um programa que utilize uma ``ArrayList`` para armazenar uma lista de filmes.

2. Adicione filmes à lista.

3. Implemente a verificação da presença de um filme específico na lista.

4. Imprima a lista de filmes e o resultado da verificação.

## Passos para Fazer o Exercício
1. Importe as classes necessárias:

    * Utilize ``ArrayList`` e ``List`` da biblioteca ``java.util``.

    * Inicialize uma ``ArrayList`` para armazenar os filmes.

    * Adicione filmes à lista.

    * Imprima a lista de filmes.

    * Verifique se um filme específico está na lista e imprima o resultado.

3. Teste a funcionalidade:

    * Execute o programa e verifique se as operações de adição, impressão e verificação estão funcionando corretamente.
